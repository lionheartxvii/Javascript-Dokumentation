Leaflet

1. Leaflet herunterladen

Auf der Website von Leaflet die Bibliothek herunterladen und im Libs Ordner entpacken:

Leaflet hier herunterladen

2. Leaflet wie folgt einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<link rel="stylesheet" href="./libs/leaflet/leaflet.css">
<script src="./libs/leaflet/leaflet.js"></script>

3. Leaflet initiieren

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.

let map = L.map('map-container', {
    fullscreenControl: true,
}).setView([49.75911, 6.63203], 15);

L.tileLayer('https://tiles.stadiamaps.com/tiles/alidade_smooth_dark/{z}/{x}/{y}{r}.png', {
    attribution: '© Stadia Maps, © OpenMapTiles © OpenStreetMap contributors'
}).addTo(map);

L.marker([49.75911, 6.63203]).addTo(map)
    .bindPopup('Irminenfreihof 8, Intermedia Design')
    .openPopup();

4. In HTML einbauen

Für die Karte muss bloß ein Container erstellt werden.

<div id="map-container"></div>

5. CSS einfügen

Die Karte durch CSS anpassen.

#map-container{
    width: 70%;
    height: 300px;
    margin: 1rem auto;
}

An dieser Stelle des Scripts lassen sich die Koordinaten sowie Zoom Faktor ändern.

.setView([49.75911, 6.63203], 15)

9. Plugin einfügen

Mit dem Plugin kann man die Karte in einem Vollbild Modus öffnen.
Unter PlugIns > Leaflet.fullscreen > Auf Github runterladen und im Libs Ordner entpacken.

Script und CSS einbinden: 

<script src="./libs/Leaflet.fullscreen-gh-pages/dist/Leaflet.fullscreen.min.js"></script>
<link rel="stylesheet" href="./libs/Leaflet.fullscreen-gh-pages/dist/leaflet.fullscreen.css">

Plugin initiieren, indem man das Script dazufügt und fullscreenControl auf true setzt. 

<script>
let map = L.map('map-container', {
fullscreenControl: true,
}).setView([49.75911, 6.63203], 15);
</script>
