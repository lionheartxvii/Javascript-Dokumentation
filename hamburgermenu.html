Hamburger Menu

1. Illustrator Datei

1.1 Eine Illustrator Datei anlegen und Datei erstellen.
1.2 SVG korrekt exportieren.
1.3 Einen div.svg-container erstellen.
1.4 Die SVG in VSC öffnen und den Code in den svg-container kopieren.
1.5 SVG bekommt die id "hamburger-icon".
1.6 CSS herauskopieren und in den head kopieren.
1.7 Stroke-miterlimit: 10; kann man löschen.
1.8 Den .svg-container, top und bottom stylen:
1.9 In der Konsole kann man testen, wie die Werte sein sollen. 

.svg-container{
    max-width: 400px;
    margin: 3rem auto;
}

#top, #bottom{
    stroke-dasharray: 36px 200px;
    stroke-dashoffset: 127;
}

2. Script für Klasse active einbauen

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.

<script>
    document.getElementById('hamburger-icon').addEventListener('click', function(event){
    this.classList.toggle('active');
    })
</script>

3. Animation erstellen

Top und Bottom wird eine Animation hinzugefügt und die Strokes werden definiert.
Auch 'active' muss definiert werden.

#top, #bottom{
    stroke-dasharray: 36px 200px;
    stroke-dashoffset: 127;
    transition: all 0.5s;
}
      
.active #top,
.active #bottom{
     stroke-dashoffset: 236;
}

Für den mittleren Balken muss die Mitte und die Rotation festgelegt werden und dann wird die Animation definiert.
Ebenfalls wird die active Version gestylt. 

#center{
    transform-origin: 50px 50px;
    transform: rotate( 0deg ) ;
    transition: all 0.5s;
}

.active #center{
    transform: rotate( 225deg ) ;
}

4. CSS Animationen Property

Durch die CSS Eigenschaften kann man weitere Eigenschaften der Animationen verändern.
Dazu muss man die transition für top und bottom verändern.
Ebenfalls kann man bereits die Keyframes definieren. 

.active #top,
.active #bottom{
    animation-name: hamburger2x;
    animation-duration: 0.5s;
    animation-fill-mode: forwards;          
}

@keyframes hamburger2x {
    0%{
        stroke-dashoffset: 127;
    }

    100%{
        stroke-dashoffset: 236;
    }
}

5. Damit die Animation auch rückwärts abspielt

Damit die Animation auch rückwärts abspielt, erstellt man die Animation x2hamburger.
Ebenfalls legt man die Klasse backwards fest. 

@keyframes x2hamburger {
    0%{
        stroke-dashoffset: 236;
    }

    100%{
        stroke-dashoffset: 127;
    }
}

.backwards #top,
.backwards #bottom{
    animation-name: x2hamburger;
    animation-duration: 0.5s;
    animation-fill-mode: forwards;
}

6. Script anpassen

<script>
    document.getElementById('hamburger-icon').addEventListener('click', function(event){
        if(this.classList.contains('active') ){
            this.classList.remove('active');
            this.classList.add('backwards');
        }else{
            this.classList.add('active');
            this.classList.remove('backwards');
        }
})

</script>
