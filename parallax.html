Scrollmagic - Parallax

1. Scrollmagic herunterladen

Auf der Website von Scrollmagic die Bibliothek herunterladen und im Libs Ordner entpacken:

Scrollmagic hier herunterladen

2. Scrollmagic wie folgt einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<script src="./libs/ScrollMagic/scrollmagic/minified/ScrollMagic.min.js"></script>
<script src="./libs/ScrollMagic/scrollmagic/minified/plugins/debug.addIndicators.min.js">

3. Scrollmagic initiieren

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.

<script>
    let controller = new Scrollmagic.controller();
</script>

4. Neue Szene erstellen

Um eine Animation mit ScrollMagic einzusetzen, muss eine Szene hinzugeügt werden.
Durch triggerElement gibt man das Element an, durch welches das Ereignis ausgelöst werden soll.
Und durch triggerHook kann man den Auslöser verschieben.
Mit .setClassToggle() kann man eine Klasse hinzufügen, mit der man die Animation auslöst. 

new ScrollMagic.Scene({
    triggerElement: '#item-2',
    triggerHook: 0.35
})
.addTo( controller )
.addIndicators({name: 'ClassToggle'})
.setClassToggle("#item-2", "active");

Parallax

1. HTML Gerüst bauen

Mit der id 'parallax werden später die Elemente im Script angesprochen.'

<div>

    <section id="parallax">
        <p class="parallaxtext">how to parallax</p>
        <div class="parallaxrahmen"></div>
    </section>

</div>

2. CSS einfügen

Mit 'background-attachment: fixed' bleibt das Hintergrundbild stehen. Mit 'background-position: center 100%' kann man das Bild verschieben, geht auch mit Pixelangaben. 

#parallax{
    background-color: #070602;
    height: 60vh;
    background-image: url(../assets/acrylpour13.jpg);
    background-attachment: fixed;
    background-repeat: no-repeat;
    background-size: cover;
    display: flex;
    align-items: center;
    justify-content: center;
}

3. Script anpassen

Für den Parallax Effekt muss man das Script wie folgt anpassen.
Durch die Variable 'verschiebung' kommt der Effekt zustande und kann angepasst werden. 

let parallaxsection = document.getElementById('parallax');
    new ScrollMagic.Scene({
        triggerElement: '#parallax',
        triggerHook: 'on enter',
        duration: '100%'
    })
    .addTo( controller )
    .addIndicators({name: 'Parallax', colorStart:'#ffffff', colorEnd:'#ffffff',  colorTrigger:'#ffffff'})
    .on("progress", function (event) {
        console.log( window.innerHeight );
        let verschiebung = event.progress * window.innerHeight * -0.5;
        parallaxsection.style.backgroundPosition = 'center ' + verschiebung + 'px';
    });
