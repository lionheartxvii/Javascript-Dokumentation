TippyJS

1. TippyJS herunterladen

Auf der Seite von Atomiks den 1. Produktions Link für popperjs kopieren: und in einem neuen Tab öffnen. Von dort den ganzen Inhalt kopieren. Die tippy.js Datei erstellen und den Inhalt dort hinein kopieren: Inhalt für PopperJS Den Vorgang mit dem 2. Link für tippyjs wiederholen:
Inhalt für TippyJS

2. TippyJS wie folgt einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<script src="./libs/tippy.js"></script>

3. TippyJS Script einfügen

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.
In diesen Code kommt der Inhalt zu den TippyTools hinein. 

<script>
    tippy('[data-tippy-content]');
</script>

4. Html Gerüst erstellen

Es sollen vier verschiedene Quadrate angesteuert werden, bei denen das TippyTool jeweils wo anders sitzt.
Die Klasse 'tippyToggle' dient dafür das richtige Element anzusteuern und 'tippyContent' ist der Inhalt, der in dem Tooltip erscheint. 

<div class="detail-boxes">
    <div class="erstebox tippyToggle"></div>
    <p class="tippyContent">
       Click to read more
   </p>
    <div class="zweitebox tippyToggle"></div>
    <p class="tippyContent">
       Click to read more
   </p>
    <div class="drittebox tippyToggle"></div>
    <p class="tippyContent">
       Click to read more
   </p>
    <div class="viertebox tippyToggle"></div>
    <p class="tippyContent">
       Click to read more
   </p>
</div>

6. CSS für Theme und Animation

Es ist möglich ein eigenes Theme und Animation für die TippyTools zu erstellen.

Hier mehr dazu 
.tippy-box[data-animation=scale][data-placement^=top]{transform-origin:bottom}.tippy-box[data-animation=scale][data-placement^=bottom]{transform-origin:top}.tippy-box[data-animation=scale][data-placement^=left]{transform-origin:right}.tippy-box[data-animation=scale][data-placement^=right]{transform-origin:left}.tippy-box[data-animation=scale][data-state=hidden]{transform:scale(.5);opacity:0}

7. Script anpassen

Auf dieser Website sind alle Optionen aufgelistet, welche man auf die TippyTools anwenden kann: TippyJS Tutorial
Mit Placement legt man fest, wo das TippyTool auftauchen soll.
Animation und Theme wurden zuvor im CSS festgelegt und hier eingebaut.
Durch die Funktion ist es möglich die verschiedenen Rechtecke einzeln anzusteuern. 

function TippySeite(className, placementsetting){
    tippy('.' + className, {
        content(reference){
           return reference.nextElementSibling.innerHTML; 
        },
        placement: placementsetting,
        animation: 'scale',
        theme: 'mystyle',
        })
    }
        
    TippySeite("erstebox", 'top-start');
    TippySeite("zweitebox", 'top-end');
    TippySeite("drittebox", 'bottom-start');
    TippySeite("viertebox", 'bottom-end');
