Chocolat

1. Chocolat herunterladen

Auf der Website von Chocolat.gitbook.io den folgenden Link öffnen und im Libs Ordner entpacken:

Chocolat hier herunterladen

2. Chocolat wie folgt einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<link rel="stylesheet" href="./libs/Chocolat-master/dist/css/chocolat.css">
<script src="./libs/Chocolat-master/dist/js/chocolat.js"></script>

3. Chocolat initiieren

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.

Chocolat(document.querySelectorAll('.chocolat-image'), {
        // options here
    });

4. Html Gerüst erstellen

Für eine Gallerie eine wird eine Div-Box mit Bildern erstellt und diese als Grid angelegt.
Die Klasse der Links muss 'Chocolat' beinhalten, damit die Bibliothek greift.
Das Bild muss als Link und als eingebettet werden. 

<div class="img-grid">
<a class="chocolat" href="./assets/acrylpour14.jpg">
<img src="./assets/acrylpour14.jpg" alt="">
</a>

<a class="chocolat" href="./assets/acrylpour12.jpg">
<img src="./assets/acrylpour12.jpg" alt="">
</a>

<a class="chocolat" href="./assets/acrylpour18.jpg">
<img src="./assets/acrylpour18.jpg" alt="">
</a>
</div

5. CSS einfügen

Das Grid in der CSS festlegen und stylen.
Das Overlay von Chocolat kann ebenfalls angepasst werden.

.img-grid{
    width: 60vw;
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    column-gap: 10px;
    row-gap: 5px;
    padding-top: 3rem;
}

img{
    max-width: 100%;
    height: auto;
}

.chocolat-overlay {
    background-color: #000;
}
