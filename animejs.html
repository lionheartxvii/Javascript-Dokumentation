AnimeJS

1. Anime.js herunterladen und einbauen

Auf Github herunterladen.
Und im HTML das Script einbauen: 

<script src="./libs/anime-master/lib/anime.min.js"></script>

2. HTML Struktur

Es sollen viele Div Elememte aneinander gereiht werden. Dafür kann man entweder eine section mit der Klasse 'grid' erstellen, viele divs einfügen und daraus ein grid bilden. Oder man erstellt ebenfalls eine section und erstellt die divs mit einem Script.

Beispiel 1 

<section class="grid">
    <div></div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
</section>

Beispiel 2

<section class="grid" id ="gridTest"></section>

4. CSS für Grid

section.grid {
    display: grid;
    gap: 10px;
    min-width: 30vw;
    min-height: 50vh;
}

section.grid>div {
    max-height: 12px;
    background-color: #00ffff;
}

5. Script hinzufügen

Durch die Funktion wird ein Grid aus vielen Divboxen erstellt. Diese werden im HTML eingesetzt und durch AnimeJS wird die Animation definiert. Außerdem kann man bestimmen wie viele Reihen und Zeilen das Grid haben soll. 


    //Funktion erzeugt ein x,y Feld an Divboxen, welche eine Animation abspielen
    //Animation selbst ist in der Funktion definiert 
    function GridAnime(x, y) {
        text = "";
        var element;
    
        //in for-schleife werden x*y divboxen geschrieben
        for (i = 0; i < (x * y); i++) {
            text += "<div></div>";
        }
        //die Divboxen werden an gebrauchter Stelle der HTML eingefügt
        element = document.getElementById("gridTest");
        element.style.gridTemplateColumns = ("repeat(" + x + ", 1fr)");
        element.innerHTML = text;
    
        //Animation mit animejs für divs
        anime({
            targets: '.grid > div',
            scale: [{
                    value: .1,
                    easing: 'easeOutSine',
                    duration: 500
                },
                {
                    value: 1,
                    easing: 'easeInOutQuad',
                    duration: 600
                },
                {
                    value: 1,
                    easing: 'easeInOutQuad',
                    duration: 700
                },
    
            ],
            backgroundColor: [{
                    value: '#E23FE3',
                    easing: 'linear',
                    duration: 500
                },
                {
                    value: '#7321df',
                    easing: 'linear',
                    duration: 600
                },
                {
                    value: '#FF0561',
                    easing: 'linear',
                    duration: 700
                },
            ],
            delay: anime.stagger(100, {
                from: 'center'
            }),
            direction: 'alternate',
            loop: true,
            easing: 'easeInOutSine',
        });
    }
    
    //Bestimmen wie viele Reihen und Zeilen das Grid haben soll
    GridAnime(20, 5);
