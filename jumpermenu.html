Scrollmagic hier herunterladen

2. Scrollmagic und Jumper wie folgt einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<link rel="stylesheet" href="./libs/autoJumperMenu.css">  
<script src="./libs/autoJumperModule.js"></script>
<script src="./libs/ScrollMagic/scrollmagic/minified/ScrollMagic.min.js"></script>
<script src="./libs/ScrollMagic/scrollmagic/minified/plugins/debug.addIndicators.min.js">

3. JumperMenu Script einfügen

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.
Falls bereits ein Controller in der Datei vorhanden ist, diese ersetzen oder individuelle Datei erstellen.
Dieser Controller dient dazu, dass die Punkte wieder kleiner werden, sobald man an ihnen vorbei scrollt. 

autoJumperMenu.init({
        targetElementsSelector: 'section',
        jumpLinkTextSelector: 'div'
})

    let controller = new ScrollMagic.Controller({globalSceneOptions: {duration: '100%'}});
    let sections = document.querySelectorAll('section');
    let jumperElements = document.querySelectorAll('#jumperMenuContainer a');

    sections.forEach( (element, index) => {
        new ScrollMagic.Scene({triggerElement: element})
        .setClassToggle(jumperElements[index], "active")
        .addTo(controller)
});

4. Html Gerüst gegebenenfalls anpassen

Alle Abschnitte als sections anlegen.
Die erste Div Box nach der section sollte den Titel der Bubble beinhalten und eine Klasse mit der man jede dieser Divs ansprechen kann.
Beispiel: 

<section id="parallax">
    <div class="JumpermenuBubbleText">Parallax</div> <!-- Für Jumpermenu -->
    <div class="parallaxtext">
        <a href="parallax.html">how to parallax</a>
    </div>
    <div class="parallaxrahmen"></div>
</section>

Bubbles anpassen

Mit der Klasse 'JumpermenuBubbleText', in der CSS den Text ausblenden damit er nicht in den Sektionen auftaucht sondern bloß in den Bubbles.
In der autoJumperMenu.css können weitere Eiegenschaften angepasst werden. 

.JumpermenuBubbleText{
    display: none;
}
