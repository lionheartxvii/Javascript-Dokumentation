Scrollmagic - SIV

1. Scrollmagic herunterladen

Auf der Website von Scrollmagic die Bibliothek herunterladen und im Libs Ordner entpacken:

Scrollmagic hier herunterladen

2. Scrollmagic wie folgt einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<script src="./libs/ScrollMagic/scrollmagic/minified/ScrollMagic.min.js"></script>
<script src="./libs/ScrollMagic/scrollmagic/minified/plugins/debug.addIndicators.min.js">

3. Scrollmagic initiieren

Den folgenden Code in einem Script oder in der Javascipt-Datei einfügen.

<script>
    let controller = new Scrollmagic.controller();
</script>

4. Neue Szene erstellen

Um eine Animation mit ScrollMagic einzusetzen, muss eine Szene hinzugeügt werden.
Durch triggerElement gibt man das Element an, durch welches das Ereignis ausgelöst werden soll.
Und durch triggerHook kann man den Auslöser verschieben.
Mit .setClassToggle() kann man eine Klasse hinzufügen, mit der man die Animation auslöst. 

new ScrollMagic.Scene({
    triggerElement: '#item-2',
    triggerHook: 0.35
})
.addTo( controller )
.addIndicators({name: 'ClassToggle'})
.setClassToggle("#item-2", "active");

3. Javascipt einbauen

ScrollMagic initiieren

let controller = new Scrollmagic.controller();

Die Szene erstellen.
Mit der Funktion 'ScrollIn' werden die Elemente einzeln angesprochen und der offsetvalue kann ebenfalls einzeln angespasst werden, wodurch die Elemente nacheinander auftauchen.
Auf der Seite von ScrollMagic befinden sich einige Beispiele, deren Code man anpassen kann.
Hier sind Beispiele 

function  ScrollIn(className, offsetvalue ){
    var revealElements = document.getElementsByClassName(className);
        for (var i=0; i<revealElements.length; i++) { 
            new ScrollMagic.Scene({
                                triggerElement: revealElements[i], 
                                offset: offsetvalue,												 
                                triggerHook: 0.92,
                                delay: 150,
                            })
                            .setClassToggle(revealElements[i], "visible") 
                            .addIndicators({name: className+ (i+1) }) 
                            .addTo(controller);
        }
}

ScrollIn("pinkesrechteck", 50);
ScrollIn("lilarechteck", 150);
ScrollIn("blauesrechteck", 250);
