Accordion

1. CSS und Javascript-Datei einbetten

Es bietet sich an, das Javascript am Ende des HTML-Dokuments (vor schließendem body-tag) zu platzieren. Es muss im Quellcode nach dem Accordion-Inhalten platziert werden.

<link rel="stylesheet" href="mylib/accordion.css">
<script src="mylib/accordion.js"></script>

2. Folgende HTML-Struktur für Accordion-Funktionalität einsetzten

4. CSS Animation einfügen

.accordion-toggle:before{
        content: '\25B8';
        display: inline-block;
        padding-right: 0.5rem;
        transition: all 0.5s;
        transform-origin: 30% 50%;
    }
    
    .accordion-toggle.active:before{
        transform: rotate(90deg);
    }
    
    .accordion-content{
        max-height: 0;
        overflow: hidden;
        transition: all 0.5s; 
    }
    
    .accordion-toggle:hover{
        cursor: pointer;
    }

Einstellung Einzel- / multipler Modus

Soll immer nur ein Inhalt entfaltet werden (nicht gewählte Inhalte werden versteckt), so muss dem ersten "accordion-toggle"-Element das Attribut data-accordionmode="single" zugewiesen werden. 

<h2 class="accordion-toggle" data-accordionmode="single">you wanna see?</h2>
<div class="accordion-content">
<p> visible/hidden content-html...</p>
</div>
