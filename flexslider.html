Flexslider

1. Flexslider herunterladen

Datei auf GitHub runterladen und im Libs Ordner entpacken:

Flexslider hier herunterladen

2. Flexslider einbinden

Die Ordnerstruktur muss für das Einbinden angepasst werden.

<script src="./libs/flexslider/jquery.flexslider-min.js"></script>
<link rel="stylesheet" href="./libs/flexslider/flexslider.css">

3. Flexslider initiieren

Flexslider einbinden, wie auf der Website gezeigt:

Flexslider Tutorial

In dem Script ist ein veraltetes jQuery drin, das wie folgt korrigiert werden muss:

<script type="text/javascript" charset="utf-8">
    $(window).on('load', function() {
    $('.flexslider').flexslider();
    });
</script>

Außerdem muss das komplette jQuery runtergeladen und eingebunden sein.

4. Html Gerüst erstellen

Der Flexslider soll zwei Bilder nebeneinander zeigen und je nachdem wie viele Bilder der Slider haben soll, fügt man diese im HTML hinzu.
Der Eltern Container muss die Klasse 'flexslider' haben und danach soll eine ul mit der Klasse 'slides' erstellt werden, in denen sich die Listen Elemente der Bilder befinden. 

<div class="flexslider">
    <ul class="slides">
      <li>
        <img src="./assets/acrylpour1.jpg" />
      </li>
      <li>
        <img src="./assets/acrylpour8.jpg" />
      </li>
      <li>
        <img src="./assets/acrylpour14.jpg" />
      </li>
      <li>
        <img src="./assets/acrylpour6.jpg" />
      </li>
    </ul>
</div>

5. CSS einfügen

Die Container stylen und den Border entfernen.

#flexslider{
    height: 100vh;
    max-width: 100vw;
    background-color: #18182c;
    display: flex;
    flex-direction: column;
    align-items: center;
    justify-content: center;
}

.flexslider{
    width: 60vw;
    height: 42vh;
    background-color: #18182c;
    border: 0px;
}

6. Script anpassen

Auf dieser Website sind alle Optionen aufgelistet, welche man auf die Slider anwenden kann: Flexslider Tutorial 

$(window).on('load', function() {
    $('.flexslider').flexslider({
      animation: "slide",
      slideshow: true,
      slideshowSpeed: 7000,
      easing: "swing",
      itemWidth: 460,
      itemMargin: 5, 
      maxItems: 2,  
      smoothHeight: true,
    });
});
